# war3-pfx-docs

Written tutorials and documentation for [PopcornFX for Warcraft III](https://war3-pfx.github.io/war3-pfx-docs).

## Related

- [war3-pfx-boilerplate](https://github.com/war3-pfx/war3-pfx-boilerplate) — ready-to-use PopcornFX project

## Local preview

```bash
pip install mkdocs-material
mkdocs serve
```

Then open [http://localhost:8000](http://localhost:8000).
